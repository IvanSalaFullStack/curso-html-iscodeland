Truco # 4 en Visual Studio Code

Si quieres hacer más grande la letra de una página web o dar zoom o quitar zoom, pulsa la tecla: CTRL junto con el símbolo + (está junto a intro). Si pulsas ambas teclas juntas varias veces la letra será cada vez más grande:
Ctrl +


En el sentido contrario, puedes hacer más pequeña la letra pulsando
CTRL junto con el símbolo de menos (o el guión normal - ):


Ctrl -


Truco # 5 en Visual Studio Code

Si escribimos una etiqueta o elemento HTML en VSC basta con solo poner el nombre de la etiqueta y dar enter:

                
                    <div> + Enter

Truco # 6 en Visual Studio Code

Para crear varias etiquetas al mismo tiempo solo tienes que escribir la etiqueta + * asterisco + 7 que sera el numero de etiqueta que deseas escribir, por ejemplo:

        
            <ul>
                <li*7>
            <ul>
y al dar click en enter tendras algo asi: 
        
            <ul>
                <li>
                <li>
                <li>
                <li>
                <li>
                <li>
                <li>
            <ul>

Truco # 7 en Visual Studio Code

El siguiente ejemplo muestra una etiqueta (section), seguida de otra etiqueta (article), que contiene un (ul) con ocho (li), para lograr esto solo tenemos que colocar:
(section>article>ul>li*8)
El nombre de la etiqueta section + el simbolo de mayor que > + el nombre de la etiqueta article + el simbolo de mayor que + el nombre de la etiqueta ul + el simbolo de mayor que + el nombre de la etiqueta li, seguida del simbolo * asterisco y el numero de etiquetas que deseas colocar
(section>article>ul>li*8) 

            
                <section>
                <article>
                        <ul>
                            <li>
                            <li>
                            <li>
                            <li>
                            <li>
                            <li>
                            <li>
                            <ul>
                <article>
                <section>
